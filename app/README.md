# Frontend Application


### Run Projects

[localhost:3000](localhost:3000)

Install Packages
```bash
yarn 

# or

npm install
```

Run
```bash
yarn start

# or

npm run start
```

### Run Project in Docker

[localhost](localhost) or [IP or Domain name]()

Build
```bash
docker-compose build
```

Run
```bash
docker-compose up -d
```
